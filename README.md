# Get-IPGeolocation.ps1
PowerShell script to perform a REST API query to get IP address geo-location information

## Parameters

- -IP
Specifies the IP address to lookup.

## Examples
```
.\Get-IPGeolocation.ps1 -IP 8.8.8.8
```
